Simple appointment booking for WordPress, with an optional Pro add-on for advanced features.

== Description ==

BookPoint is a lightweight appointment booking plugin for WordPress.

Core features included in the free version:

* Services & Categories
* Booking widget (shortcode + block)
* Calendar & schedule basics
* Customers & bookings management

Optional Pro add-on unlocks additional features (sold separately):

* Locations
* Service Extras
* Promo Codes
* Holidays management
* Payments

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install via the WordPress Plugins screen.
2. Activate the plugin.
3. Add the booking widget using the block or shortcode.

== Frequently Asked Questions ==

= Does the free version work without a license? =

Yes. The free version is fully usable without a license. Pro features require the Pro add-on.

= How do I upgrade to Pro? =

Purchase a plan from the pricing page, download the Pro ZIP, then install it via **Plugins -> Add New -> Upload Plugin** and activate it.
